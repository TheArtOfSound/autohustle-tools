Setup Guide

  1. Download — Click the download link above and choose your platform (Windows or Mac).
  2. Unzip — Extract the downloaded .zip file to a folder on your computer.
  3. Run Setup — Double-click setup.bat (Windows) or run ./setup.sh (Mac/Linux). This installs Python dependencies automatically.
  4. Activate License — Run python activate.py and paste your license key from above.
  5. Start AutoHustle — Double-click start.bat (Windows) or run ./start.sh (Mac/Linux).
  6. Open Dashboard — Go to http://localhost:8888 in your browser.
  7. Configure API Keys — In Settings, add your free Groq API key and GitHub token.
