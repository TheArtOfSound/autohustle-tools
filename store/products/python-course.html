Python Basics & Setup

Setting Up Your Environment

Before writing any code, you need Python installed on your machine. Head to python.org and download the latest version of Python 3 (3.11 or newer is ideal). During installation on Windows, make sure to check the box that says "Add Python to PATH" -- this is critical for running Python from the command line.

After installation, verify it works by opening a terminal (Command Prompt on Windows, Terminal on Mac) and typing:

python --version
# Should output something like: Python 3.12.1

We also recommend installing an editor. VS Code (Visual Studio Code) is free and has excellent Python support. Install the Python extension from the Extensions marketplace for features like syntax highlighting, code completion, and integrated debugging.

Variables and Data Types

Variables are how you store information in Python. You do not need to declare a type -- Python figures it out automatically. Here are the fundamental data types you will work with every day:

# Strings - text data wrapped in quotes
name = "AutoHustle"
greeting = 'Hello, World!'

# Integers - whole numbers
user_count = 1500
year = 2026

# Floats - decimal numbers
price = 29.99
tax_rate = 0.08

# Booleans - True or False
is_active = True
has_paid = False

# Lists - ordered collections
tools = ["json-formatter", "word-counter", "password-gen"]

# Dictionaries - key-value pairs
user = {
    "name": "Alex",
    "email": "alex@example.com",
    "plan": "pro"
}

You can check a variable's type using the built-in type() function:

print(type(name))      # <class 'str'>
print(type(user_count)) # <class 'int'>
print(type(price))     # <class 'float'>
print(type(tools))     # <class 'list'>

Functions

Functions let you group code into reusable blocks. You define a function with def, give it a name, and optionally accept parameters:

# Basic function
def greet(name):
    return f"Hello, {name}! Welcome to the course."

message = greet("Sarah")
print(message)  # Hello, Sarah! Welcome to the course.

# Function with default parameter
def calculate_discount(price, discount=0.10):
    savings = price * discount
    final = price - savings
    return final

print(calculate_discount(100))       # 90.0  (10% off)
print(calculate_discount(100, 0.25)) # 75.0  (25% off)

Control Flow

Control flow statements let your code make decisions and repeat actions:

# If/elif/else - making decisions
temperature = 72

if temperature > 85:
    print("It's hot outside!")
elif temperature > 60:
    print("Nice weather today.")
else:
    print("Bring a jacket.")

# For loops - iterating over collections
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}")

# While loops - repeating until a condition is met
countdown = 5
while countdown > 0:
    print(f"Launching in {countdown}...")
    countdown -= 1
print("Liftoff!")

Reading and Writing Text Files

Python makes file operations straightforward. The with statement ensures files are properly closed after you are done with them, even if an error occurs:

# Writing to a file
with open("output.txt", "w") as f:
    f.write("Line 1: Hello from Python!\n")
    f.write("Line 2: Automation is powerful.\n")

# Reading a file
with open("output.txt", "r") as f:
    content = f.read()
    print(content)

# Reading line by line (memory efficient for large files)
with open("output.txt", "r") as f:
    for line in f:
        print(line.strip())  # .strip() removes newline chars

Working with CSV Files

CSV (Comma-Separated Values) files are one of the most common data formats you will encounter. Python's built-in csv module handles them well:

Batch File Operations

The os and pathlib modules let you work with the filesystem -- listing directories, renaming files, and organizing data automatically:

Working with REST APIs

REST APIs are the backbone of modern software. They use standard HTTP methods -- GET to read, POST to create, PUT to update, DELETE to remove:

import requests
import json

BASE_URL = "https://jsonplaceholder.typicode.com"

# GET - Retrieve data
response = requests.get(f"{BASE_URL}/posts/1")
post = response.json()
print(f"Title: {post['title']}")

# POST - Create new data
new_post = {
    "title": "Automation with Python",
    "body": "Learn to automate your workflow...",
    "userId": 1
}
response = requests.post(f"{BASE_URL}/posts", json=new_post)
print(f"Created post ID: {response.json()['id']}")

# Headers and Authentication
headers = {
    "Authorization": "Bearer your_api_key_here",
    "Content-Type": "application/json"
}
response = requests.get(f"{BASE_URL}/posts", headers=headers)

# Error handling for API calls
try:
    response = requests.get(f"{BASE_URL}/posts/1", timeout=10)
    response.raise_for_status()  # Raises exception for 4xx/5xx
    data = response.json()
except requests.exceptions.Timeout:
    print("Request timed out. Try again later.")
except requests.exceptions.HTTPError as e:
    print(f"HTTP Error: {e}")
except requests.exceptions.RequestException as e:
    print(f"Request failed: {e}")

Building a Practical Scraper

Here is a more complete example that scrapes product data and saves it to a CSV file, with proper error handling and rate limiting:

import requests
from bs4 import BeautifulSoup
import csv
import time

def scrape_products(url):
    """Scrape product data from a page."""
    try:
        response = requests.get(url, timeout=15)
        response.raise_for_status()
    except requests.exceptions.RequestException as e:
        print(f"Failed to fetch {url}: {e}")
        return []

    soup = BeautifulSoup(response.text, "html.parser")
    products = []

    for card in soup.select(".product-card"):
        name = card.select_one(".product-name")
        price = card.select_one(".product-price")
        if name and price:
            products.append({
                "name": name.text.strip(),
                "price": price.text.strip()
            })

    return products

# Scrape multiple pages with delay
all_products = []
for page in range(1, 6):
    url = f"https://example.com/products?page={page}"
    products = scrape_products(url)
    all_products.extend(products)
    print(f"Page {page}: found {len(products)} products")
    time.sleep(2)  # Be polite: wait 2 seconds between requests

# Save results
with open("products.csv", "w", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=["name", "price"])
    writer.writeheader()
    writer.writerows(all_products)

print(f"Saved {len(all_products)} products to products.csv")

Sending Emails with Python

Python's smtplib and email modules handle email sending. Here is how to send a professional HTML email:

import smtplib
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart

def send_email(to_email, subject, html_body):
    # SMTP configuration (using Gmail as example)
    smtp_server = "smtp.gmail.com"
    smtp_port = 587
    sender_email = "your_bot@gmail.com"
    sender_password = "your_app_password"  # Use App Passwords, not your real password

    # Build the message
    msg = MIMEMultipart("alternative")
    msg["Subject"] = subject
    msg["From"] = sender_email
    msg["To"] = to_email

    html_part = MIMEText(html_body, "html")
    msg.attach(html_part)

    # Send it
    try:
        with smtplib.SMTP(smtp_server, smtp_port) as server:
            server.starttls()  # Encrypt the connection
            server.login(sender_email, sender_password)
            server.send_message(msg)
            print(f"Email sent to {to_email}")
    except Exception as e:
        print(f"Failed to send email: {e}")

# Usage
send_email(
    "boss@company.com",
    "Daily Sales Report - Jan 15",
    "<h2>Sales Report</h2><p>Total revenue today: <strong>$1,247.50</strong></p>"
)
Note: For Gmail, you need to create an App Password in your Google Account settings. Never put your actual password in code. Use environment variables or a config file for credentials.

Building an Automated Report System

Combining file processing, data analysis, and email delivery into a complete reporting pipeline:

import csv
import json
from datetime import datetime, timedelta

def generate_report():
    """Read sales data and generate a summary."""
    with open("sales.csv", "r") as f:
        reader = csv.DictReader(f)
        rows = list(reader)

    total_revenue = sum(float(r["Revenue"]) for r in rows)
    total_orders = len(rows)
    avg_order = total_revenue / total_orders if total_orders > 0 else 0

    # Find best seller
    from collections import Counter
    products = Counter(r["Product"] for r in rows)
    best_seller = products.most_common(1)[0] if products else ("N/A", 0)

    report = {
        "date": datetime.now().strftime("%Y-%m-%d"),
        "total_revenue": round(total_revenue, 2),
        "total_orders": total_orders,
        "average_order_value": round(avg_order, 2),
        "best_seller": best_seller[0],
        "best_seller_count": best_seller[1]
    }

    # Save as JSON
    with open("daily_report.json", "w") as f:
        json.dump(report, f, indent=2)

    # Build HTML for email
    html = f"""
    <h2>Daily Report - {report['date']}</h2>
    <table>
        <tr><td>Total Revenue</td><td><strong>${report['total_revenue']}</strong></td></tr>
        <tr><td>Total Orders</td><td>{report['total_orders']}</td></tr>
        <tr><td>Avg Order Value</td><td>${report['average_order_value']}</td></tr>
        <tr><td>Best Seller</td><td>{report['best_seller']} ({report['best_seller_count']} sales)</td></tr>
    </table>
    """
    return html

# Generate and email the report
html = generate_report()

Step 3: The Scraper Module

# scraper.py
import requests
from bs4 import BeautifulSoup
import json

def load_sources():
    with open("sources.json", "r") as f:
        return json.load(f)

def scrape_rss(url):
    """Parse an RSS feed and return articles."""
    try:
        response = requests.get(url, timeout=15)
        response.raise_for_status()
        soup = BeautifulSoup(response.text, "xml")

        articles = []
        for item in soup.find_all("item")[:5]:
            articles.append({
                "title": item.title.text if item.title else "Untitled",
                "link": item.link.text if item.link else "",
                "summary": item.description.text[:200] if item.description else ""
            })
        return articles
    except Exception as e:
        print(f"Error scraping {url}: {e}")
        return []

def gather_all_content():
    """Scrape all configured sources."""
    sources = load_sources()
    all_content = {}
    for source in sources:
        print(f"Scraping: {source['name']}...")
        articles = scrape_rss(source["rss_url"])
        all_content[source["name"]] = articles
    return all_content

Step 4: The Email Module

# emailer.py
import smtplib
import json
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart
from datetime import datetime

def load_config():
    with open("config.json", "r") as f:
        return json.load(f)

def build_digest_html(content):
    """Build an HTML email from aggregated content."""
    date = datetime.now().strftime("%B %d, %Y")
    html = f"<h1>Daily Digest - {date}</h1>"

    for source_name, articles in content.items():
        html += f"<h2>{source_name}</h2>"
        if not articles:
            html += "<p><em>No articles today.</em></p>"
            continue
        for a in articles:
            html += f"""
            <div style='margin-bottom:12px'>
                <a href='{a["link"]}'><strong>{a["title"]}</strong></a>
                <p>{a["summary"]}</p>
            </div>"""

    return html

def send_digest(content):
    """Send the digest email."""
    config = load_config()["email"]
    html = build_digest_html(content)

    msg = MIMEMultipart("alternative")
    msg["Subject"] = f"Your Daily Digest - {datetime.now().strftime('%b %d')}"
    msg["From"] = config["sender"]
    msg["To"] = ", ".join(config["recipients"])
    msg.attach(MIMEText(html, "html"))

    with smtplib.SMTP(config["smtp_server"], config["smtp_port"]) as server:
        server.starttls()
        server.login(config["sender"], config.get("password", ""))
        server.send_message(msg)
    print("Digest sent successfully!")

Step 5: The Main Bot

# bot.py
import schedule
import time
import json
from scraper import gather_all_content
from emailer import send_digest

def run_daily_digest():
    """Main bot function: scrape, build, send."""
    print("Starting daily digest generation...")
    content = gather_all_content()

    # Count total articles
    total = sum(len(articles) for articles in content.values())
    print(f"Collected {total} articles from {len(content)} sources.")

    # Send the digest
    send_digest(content)
    print("Daily digest complete!")

def main():
    config = json.load(open("config.json"))
    send_time = config["settings"]["send_time"]

    print(f"Content Bot started. Digest scheduled for {send_time} daily.")
    schedule.every().day.at(send_time).do(run_daily_digest)

    # Also run once immediately on start
    run_daily_digest()

    while True:
        schedule.run_pending()
        time.sleep(60)

if __name__ == "__main__":
    main()

  • PythonAnywhere — Free tier supports scheduled tasks, perfect for beginners. Upload your files and set up a scheduled task to run bot.py.
  • GitHub Actions — Free for public repos. Create a workflow file to run your script on a schedule.
