Getting Started

Base URL

https://api.autohustle.online/v1

Authentication

All requests require an API key sent as a Bearer token in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Response Format

All responses are returned in JSON format. Successful responses include a status field set to "ok". Error responses include a status of "error" and a human-readable message field.

Rate Limits

1,000
Requests per day
60
Requests per minute

Rate limit headers are included in every response: X-RateLimit-Remaining and X-RateLimit-Reset.

Status Codes

200Success 400Bad Request — invalid parameters 401Unauthorized — invalid or missing API key 429Rate Limited — too many requests 500Server Error — try again later

Inspirational Quotes API

Access a curated collection of over 10,000 inspirational quotes from thought leaders, authors, entrepreneurs, and philosophers. Perfect for apps, websites, bots, and digital signage.

GET /quotes/random

Returns a single random quote. Optionally filter by category or author.

ParameterTypeDescription
categorystring optionalFilter by category: motivation, success, wisdom, life, business, creativity, leadership, perseverance
languagestring optionalResponse language (default: en). Supported: en, es, fr, de, pt
Example Request
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.autohustle.online/v1/quotes/random?category=motivation"
Example Response
{
  "status": "ok",
  "data": {
    "id": "q_8f3a2b1c",
    "text": "The only way to do great work is to love what you do.",
    "author": "Steve Jobs",
    "category": "motivation",
    "tags": ["work", "passion", "success"],
    "source": "Stanford Commencement Speech, 2005"
  }
}

Text Analysis API

Analyze any text for readability, sentiment, keyword density, word count, and more. Ideal for content tools, writing assistants, SEO analysis, and editorial workflows.

POST /text/readability

Returns only readability scores and grade-level analysis. Faster than the full analyze endpoint when you only need readability data.

Body FieldTypeDescription
textstring requiredThe text to analyze (max 50,000 characters)
target_gradeinteger optionalTarget reading grade level (1-16). Returns suggestions if text exceeds target.
Example Request
curl -X POST "https://api.autohustle.online/v1/text/readability" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Your text here...",
    "target_grade": 8
  }'

Quick Integration Examples

JavaScript / Node.js

const API_KEY = 'your_api_key_here';
const BASE = 'https://api.autohustle.online/v1';

// Get a random quote
async function getRandomQuote(category = '') {
  const url = category
    ? `${BASE}/quotes/random?category=${category}`
    : `${BASE}/quotes/random`;

  const res = await fetch(url, {
    headers: { 'Authorization': `Bearer ${API_KEY}` }
  });
  const json = await res.json();
  return json.data;
}

// Analyze text
async function analyzeText(text) {
  const res = await fetch(`${BASE}/text/analyze`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ text })
  });
  const json = await res.json();
  return json.data;
}

Python

import requests

API_KEY = 'your_api_key_here'
BASE = 'https://api.autohustle.online/v1'
HEADERS = {'Authorization': f'Bearer {API_KEY}'}

# Get a random quote
def get_random_quote(category=None):
    params = {'category': category} if category else {}
    r = requests.get(f'{BASE}/quotes/random', headers=HEADERS, params=params)
    return r.json()['data']

# Analyze text
def analyze_text(text):
    r = requests.post(
        f'{BASE}/text/analyze',
        headers={**HEADERS, 'Content-Type': 'application/json'},
        json={'text': text}
    )
    return r.json()['data']

# Example usage
quote = get_random_quote('motivation')
print(f'"{quote["text"]}" - {quote["author"]}')

analysis = analyze_text('Your text to analyze here.')
print(f'Readability: Grade {analysis["readability"]["flesch_kincaid_grade"]}')
print(f'Sentiment: {analysis["sentiment"]["overall"]}')
